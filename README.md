# community-service-server
基于SpringBoot+Vue前后端分离的社区服务平台网系统（源码+文档），系统包含管理员、用户角色，功能如下： 管理员：首页统计（预约统计、帖子统计）；用户管理；活动管理；活动报名；论坛管理（论坛版块、帖子管理、评论管理）；服务管理；服务预约；服务评价；新闻管理；留言管理；公告信息；个人信息；密码修改； 用户：注册、登录系统；首页；社区活动（活动列表、详情、报名、报名名单）；社区论坛（版块、帖子、评论、点赞、收藏）；社区服务（服务列表、详情、预约、评价）；社区新闻；系统留言；公告信息。用户还具有个人中心功能，功能包括：活动报名、我的帖子、帖子评论、帖子点赞、帖子收藏、服务预约、服务评价、我的留言、个人中心、修改密码。
